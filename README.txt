O projeto consiste em enviar requisições via wi-fi para o módulo
ESP32 através do aplicativo desenvolvido na plataforma MIT APP
INVENTOR, ao pressionar e liberar algum botão no aplicativo uma
requisição é enviada ao ESP32, a partir daí ele envia sinais de controle para
a ponte H (l298n), que além de permitir definir o sentido de rotação do eixo
dos motores, também possibilita acionar os dois ao mesmo tempo ou
somente um deles permitindo que o carrinho faça curvas.
